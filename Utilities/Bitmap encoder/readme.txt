You have to download Processing to run this encoder : https://www.processing.org/download/

== How to use ==
1. Put a picture named "input.bmp" in the same folder than "Bitmpap encoder.pde"
2. Open "Bitmap encoder.pde" with Processing
3. Run the program. It will display the original picture,
   and the picture after the monochrome converstion. Then close it.
4. It created a file named "output.txt". Open it, copy and paste the content into your Gamebuino sketch.
5. Don't forget to change "bitmapName" to whatever you want.

For more info visit http://gamebuino.com/wiki/index.php?title=Bitmap_encoder
